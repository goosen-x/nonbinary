## 1. Open-Meteo (Погода)

**URL**: https://open-meteo.com
**API ключ**: Не требуется
**Лимиты**: 10,000 запросов/день

```
https://api.open-meteo.com/v1/forecast?latitude=55.75&longitude=37.62&current_weather=true
```

### Проект: Погодный виджет с прогнозом

Создать интерактивное веб-приложение для отображения погоды с автоопределением геолокации пользователя. Приложение показывает текущую температуру, влажность, скорость ветра и прогноз на 7 дней в виде графиков. Дополнительно можно добавить переключение между городами и сохранение избранных локаций в localStorage.

**Сложность**: Начальная (2/5)

**Трудности**:
- Обработка ошибок геолокации (пользователь может запретить доступ)
- Конвертация координат в название города (нужен обратный геокодинг через Nominatim)
- Адаптивная вёрстка для мобильных устройств

**Установка**:
```bash
npm create vite@latest weather-app -- --template react-ts
npm install axios recharts date-fns
```

**Реализация**:
1. Получить координаты через `navigator.geolocation.getCurrentPosition()`
2. Сделать запрос к Open-Meteo с координатами
3. Отрисовать данные с помощью Recharts (графики) и CSS Grid (карточки)
4. Добавить поиск города через Nominatim API
5. Сохранить избранные города в localStorage
